Applications

  • We develop multi-platform apps that aren't bound to one type of software, They can be built for Windows, MacOS, IOS and Android.
  • Native apps are developed upon request.
  • Framework used: .Net Maui by Microsoft.
  • Languages used: XAML, C#.

Web-Applications

  • Web apps are developped in one of those two ways:
    • The first one is by making web apps that can run offline and have an icon that can be added to the home screen.
    • The other option is faster and more secure but will not have the options of the former one.
  • Web apps can have the same features as a regular app and more.
  • Framework used: .Net Blazor by Microsoft.
  • Languages used: HTML5, CSS3, Javascript, C#.

Websites

  • Websites are built in three options:
    • The first is a one of a kind website that will have a lot thought put into the design to make it as unique as possible.
    • The second is a regular website with the design being a modern one and an expected amount of work on the design is done. Great option for startups.
    • The third is developed with a design that can be found in many places, not special but enough for having a nice place where people can search you, perfect for a low budget.
  • Framework used: .Net Blazor by Microsoft.
  • Content Management System: Wordpress.
  • Languages used: HTML5, CSS3, Javascript, C#.
